### vscode usage guide

- cut line

    `Ctrl+X`

- new file

    `Ctrl+N`

- save file

    `Ctrl+S`

- open terminal

    `Ctrl+Shift+backtick`

- open preview

    `Ctrl+K V`

- show all commands

    `Ctrl+Shift+P`

- find in file

    `Ctrl+Shift+F`

- switch to file

    `Ctrl+P`
